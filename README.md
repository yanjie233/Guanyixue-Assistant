# 莞易学助手 (Guanyixue Assistant)

**莞易学助手** 是一款专为 `study.dgjy.net` 平台设计的浏览器扩展插件，旨在提供快捷、自动化的学习体验，大幅提升学习效率。

- **版本**: 3.2
- **作者**: yanjie233
- **发布日期**：2026/4/16
- **更新内容**：首次发布。

---

## ✨ 核心功能

1. **🚀 智能最高16倍速播放**
   - 视频播放过程中自动开启 **16倍速** 冲刺。
   - 当视频剩余时间小于 25 秒时，自动降速至 **1.0倍速** 以确保平台记录完整进度并平稳结束。

2. **🤖 自动化课程学习 (自动连播)**
   - 自动扫描并识别 `进度未完成` 或 `未开始` 的课程。
   - 学习完当前视频后，若 60 秒内无新内容，会自动返回首页并无缝进入下一个未学习科目。

3. **🔇 专注模式 (自动静音 & 后台播放)**
   - 自动将播放的视频静音，学习不打扰正常工作。
   - 破除失焦暂停限制（解除网页 visibility 检测），允许**后台挂机播放**，切换到其他网页或软件视频依然不会暂停。

4. **📊 实时悬浮面板与弹窗面板**
   - **网页右侧悬浮窗**：实时显示当前运行状态、当前倍速、视频真实剩余时间及16倍速下的预估完成时间。
   - **扩展程序弹窗 (Popup)**：点击浏览器右上角插件图标，可一键读取你在该平台的累积 `studyTime` (学习时长) 并自动换算成 `时:分:秒` 的直观格式。

---

## 🛠️ 文件目录结构

- `manifest.json`: Chrome 扩展程序配置文件，声明了插件权限、注入脚本及版本信息。
- `content.js`: 核心内容脚本。注入到学习页面内部，包含了 UI 面板创建、16倍速调节、视频监控及课程进度自动识别与跳转的核心逻辑。
- `popup.html`: 插件扩展弹窗界面的 HTML 结构。
- `popup.js`: 插件扩展弹窗的交互逻辑，负责通过 API 发送消息并获取换算后的“总学习时长”。

## 💻 安装与使用说明

1. **下载或克隆本仓库代码** 到本地文件夹，例如 `Guanyixue Assistant`。
2. 打开 Chrome (或 Edge) 浏览器，在地址栏输入 `chrome://extensions/` 并在右上角开启 **“开发者模式”**。
3. 点击左上角 **“加载已解压的扩展程序”**，选择当前所在的文件夹即可完成安装。
4. 打开学习网站 `https://study.dgjy.net/` 并登录。
5. 脚本将自动在右上角调出悬浮面板，全自动处理视频倍速和课程跳转，您只需将其挂在后台即可！
6. 点击浏览器右上角的 **“莞易学助手”插件图标**，可查看自己的平台总学习时长。
